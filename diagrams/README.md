# WattWatch System Diagrams

This directory contains the visual system schemas for the hackathon documentation:

1. **System Architecture Diagram** (`system-architecture.png` or `system-architecture.svg`):
   Visualizes the structural connection between the IoT Simulator/Wokwi prototypes, the Node.js + Express backend server, the React dashboard client, and the Discord bot with Gemini integrations.

2. **Data Flow Diagram** (`data-flow.png` or `data-flow.svg`):
   Visualizes how device events flow from state changes, through Socket.IO real-time broadcasts and REST API endpoints, to update UI widgets and trigger proactive Discord alert broadcasts.
