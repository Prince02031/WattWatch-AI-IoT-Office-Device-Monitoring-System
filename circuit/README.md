# ESP32 Circuit Simulation

This folder contains documentation and files for the ESP32 circuit simulation on Wokwi.

## Hardware Configuration (15 Devices Simulation)
The office setup consists of exactly 15 devices across 3 rooms. In the real Wokwi ESP32 circuit simulation, we map selected indicators or use shift registers (e.g. 74HC595) or I2C port expanders (e.g. MCP23017) to drive the 15 status LEDs representing:
- **Drawing Room**: 2 Fan LEDs, 3 Light LEDs
- **Work Room 1**: 2 Fan LEDs, 3 Light LEDs
- **Work Room 2**: 2 Fan LEDs, 3 Light LEDs
